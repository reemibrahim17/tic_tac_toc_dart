# 🎮 Tic-Tac-Toe Dart Console Game

This is a simple two-player Tic-Tac-Toe game built with Dart for the console.

## 🚀 How to Run

1. Ensure Dart is installed on your system.
2. Clone the repository or download the source code.
3. Navigate to the project folder in your terminal and run:

```bash
dart run

Two-player turn-based gameplay
